# Расчетно-графическая работа по Программированию

## Тема: Разработка криптографического программного обеспечения
Криптографический проект (РГР), выполненный в рамках курса **«Программирование»**.

Целью работы является реализация консольной программы на **C++**, которая предоставляет пользователю меню для выбора и выполнения криптографических операций, включая работу с файлами.

---

### Реализованные алгоритмы

Программа реализует меню со следующими функциями:

1.  **Демонстрация Протокола Диффи-Хеллмана (DH)**
2.  **Шифрование/Дешифрование с помощью шифра Скиталы**
3.  **Шифрование/Дешифрование с помощью блочного шифра TEA**
4.  **Печать содержимого файла на экран**
5.  Выход

### Инструкция по запуску

Проект написан на **C++**. Для компиляции и запуска требуется компилятор (например, GCC/G++).

1.  **Клонирование репозитория:**
    ```bash
    git clone [https://github.com/frezqy/Crypto_Project.git]
    ```
2.  **Компиляция:**
    ```bash
    g++ main.cpp [другие .cpp файлы] -o crypto_rgr
    ```
3.  **Запуск:**
    ```bash
    ./crypto_rgr
    ```
    *Программа запустит интерактивное меню в консоли.*
