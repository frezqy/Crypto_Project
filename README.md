# Расчетно-графическая работа по Программированию

## Тема: Разработка криптографического программного обеспечения
Криптографический проект (РГР), выполненный в рамках курса **«Программирование»** (НТГУ).

Целью работы является реализация консольной программы на **C++**, которая предоставляет пользователю меню для выбора и выполнения криптографических операций.

---

### Реализованные алгоритмы

Программа реализует меню с выбором следующих алгоритмов шифрования и дешифрования:

1.  **Шифр Вижинера**
2.  **Шифр Бекона**
3.  **Одиночная перестановка по ключу**
4.  Выход

### Инструкция по запуску

Проект написан на **C++**. Для компиляции и запуска требуется компилятор (например, GCC/G++).

1.  **Клонирование репозитория:**
    ```bash
    git clone [Вставьте сюда HTTPS ссылку вашего репозитория]
    ```
2.  **Компиляция (пример для Linux/MacOS/Windows с MinGW):**
    ```bash
    g++ main.cpp -o crypto_rgr
    ```
3.  **Запуск:**
    ```bash
    ./crypto_rgr
    ```
    *Программа запустит интерактивное меню в консоли.*
